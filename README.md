# WayMont

Страница-инструкция по установке и настройке подключения на всех популярных платформах.

## Возможности

- Поддержка iOS, Android, Windows, macOS, Linux и Smart TV
- Минималистичный интерфейс без рекламы и трекеров

## Использование

Откройте сайт в браузере и следуйте инструкции:  
[https://mamontosik.github.io/waymont/](https://mamontosik.github.io/waymont/)

## Стек

- HTML + CSS (CSS Custom Properties, медиазапросы)
- Vanilla JavaScript (IntersectionObserver, lightbox)
- WebP + PNG для изображений
- Без сторонних зависимостей и скриптов

Приятного использования!
